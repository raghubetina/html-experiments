# Rails Template

This is a base Ruby on Rails repository configured for learning with Gitpod.

- Ruby version: `3.1.2`
- Rails version: `7.0.4`


We've added additional Ruby gems and other software to the `Dockerfile` that aren't automatically available in a new Rails app.
